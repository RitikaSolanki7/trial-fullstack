# trial Fullstack Project (Demo)
This archive contains a full-stack project satisfying the trial assignment.

Structure:
- backend/ : Node.js + Express + MongoDB (Mongoose)
- frontend/: React (Vite) single-page app

Features implemented:
- Landing page fetching Projects & Clients from backend
- Contact form submits to backend (visible in admin)
- Newsletter subscription
- Admin panel to add Projects and Clients (image upload + server-side cropping)
- Basic styling and responsive layout

See backend/README.md and frontend/README.md for running instructions.
