# trial Backend
Node/Express backend with MongoDB (Mongoose).
Features:
- CRUD endpoints for Projects and Clients (image upload + cropping using sharp).
- Contact form submissions and newsletter subscriptions.

How to run:
1. Install dependencies: `npm install`
2. Create `.env` from `.env.example` and set MONGO_URI if needed.
3. Start server: `npm run dev` (requires nodemon) or `npm start`
